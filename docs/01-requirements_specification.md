# Спецификация требований: Веб-сайт UWS

**Версия:** 1.0  
**Дата:** 2026-02-06

---

## 1. Введение

### 1.1 Цель проекта
Создание современного веб-сайта компании UWS (Ukrainian Window Systems) для предоставления информации об услугах установки и ремонта окон в Киеве.

### 1.2 Область применения
- Веб-сайт: uws.com.ua
- Целевая аудитория: жители Киева и Киевской области

### 2.3 Калькулятор

#### Параметры конфигурации окон

**Тип конструкции:**
- Одностворчатое окно
- Двустворчатое окно
- Трёхстворчатое окно
- Балконный блок
- Входная дверь ПВХ

**Размеры:**
- Ширина: 30-300 см
- Высота: 30-300 см

**Профильные системы:**
| Профиль | Камеры | Ширина | Цена/м² |
|---------|--------|--------|---------|
| Economy | 3 | 60мм | 2800 грн |
| Standard | 5 | 70мм | 3500 грн |
| Premium | 6 | 80мм | 4200 грн |
| Elite | 7 | 86мм | 5000 грн |

**Стеклопакеты:**
- 1-камерный 24мм
- 2-камерный 32мм
- Энергосберегающий 40мм

**Фурнитура:**
- Roto NT (Premium)
- MACO (Premium)
- Siegenia TITAN (Standard)
- Elementis (Economy)

## 3. Нефункциональные требования

### 3.1 Производительность

| Метрика | Целевое значение |
|---------|------------------|
| LCP | ≤ 2.5 сек |
| FID | ≤ 100 мс |
| CLS | ≤ 0.1 |
| PageSpeed | ≥ 90/100 |

### 3.2 Браузерная поддержка
- Chrome (последние 2 версии)
- Firefox (последние 2 версии)
- Safari (последние 2 версии)
- Edge (последние 2 версии)

### 3.3 Адаптивность
- Mobile-first подход
- Breakpoints: 320px, 480px, 768px, 1024px, 1280px, 1920px

### 3.4 Доступность
- WCAG 2.1 уровень AA
- Клавиатурная навигация
- Скринридер-совместимость
